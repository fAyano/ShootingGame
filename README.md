## 遊び方  
1. 初めの画面を読んだらenterを押す→ゲームスタート  
2. 発射台は左右の矢印キーで動かすことができる  
3. スペースキーを押すと爆弾を打つことができ、上から落ちてくる的に爆弾があたればポイントがもらえる
4. 最後にポイントが表示されて終了
